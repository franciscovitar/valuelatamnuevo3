# Value Latam Next + Sass

Migracion fiel del prototipo `value-latam-cinematic-prototype.html` a Next.js App Router + Sass.

## Comandos

```bash
npm install
npm run dev
```

La app renderiza el mismo markup del HTML final, importa los estilos globales desde `app/value-latam.scss` y ejecuta los scripts originales desde el componente cliente `components/ValueLatamClient.jsx`.
